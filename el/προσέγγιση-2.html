Θεραπευτική Προσέγγιση

Ως Προσωποκεντρική Ψυχοθεραπεύτρια, πιστεύω πως σε κάθε άνθρωπο αξίζει μια θεραπεία σχεδιασμένη στη μοναδικότητα των εμπειριών του και τον ιδιαίτερο τρόπο του να απαντά σε αυτές. Θα πρέπει να είναι μια ασφαλής, διακριτική και αποτελεσματική θεραπευτική προσέγγιση, η οποία να σέβεται το δικαίωμά του να ελέγχει πλήρως τη θεραπευτική διαδικασία.

Κάθε άνθρωπος είναι διαφορετικός, και δεν υπάρχουν τυποποιημένες λύσεις που να λειτουργούν για όλους. Ο καθένας έχει τη δική του ιδιαίτερη πορεία στην προσωπική ανάπτυξη, και ο ψυχοθεραπευτής είναι εκεί για να συνοδεύσει τον πελάτη σε αυτή τη διαδικασία: να τον βοηθήσει να ακούσει την εσωτερική του φωνή, να ανακαλύψει, να αναγνωρίσει και να ενισχύσει τα δυνατά του σημεία, και να τον βοηθήσει στην εκκίνηση αλλαγών στις προβληματικές πλευρές. Ο ρόλος μου δεν είναι να σας πω τι να κάνετε. Θα είμαι εκεί για σας βοηθήσω να βρείτε τις δικές σας απαντήσεις, αυτές που πηγάζουν από την εσωτερική σας σοφία.

Ως ακροατής, θα είμαι εκεί με σεβασμό, συμπόνια και αποδοχή να ακούσω ό,τι έχει ανάγκη να μοιραστεί ο πελάτης, χωρίς να κρίνω ή να αξιολογώ. Θα καταβάλω κάθε προσπάθεια ώστε να χτιστεί μια άνετη, ασφαλής και ζεστή ατμόσφαιρα, μέσα στην οποία θα αναπτυχθεί μια αξιόπιστη και γερή θεραπευτική σχέση. Επιπλέον, προκειμένου να αποκαλύψουμε αυτά που βρίσκονται κάτω από την επιφάνεια και προσπαθούν να αναδυθούν, θα παρατηρώ προσεκτικά τη γλώσσα του σώματος και όλα τα μη-λεκτικά σήματα που συνοδεύουν όσα διηγείται ο πελάτης. Υπάρχουν πολλά πράγματα, όπως τραύματα ή βαθιά κρυμμένες μνήμες και συναισθήματα, τα οποία είναι βωβά και μπορούν να προσεγγιστούν μόνο από τις μη-λεκτικές εκφράσεις τους.

Συμβουλευτική / Ψυχοθεραπεία

Προσφέρω τόσο Συμβουλευτική όσο και Ψυχοθεραπεία. Οι διαφορές αφορούν κυρίως τη διάρκεια της θεραπευτικής παρέμβασης, και το πόσο βαθιά προσεγγίζονται ορισμένα θέματα. Συνήθως η Συμβουλευτική επιτρέπει την ταχεία και συγκεκριμένη θεραπεία σημειακών ζητημάτων ή δυσκολιών, ενώ η ψυχοθεραπεία επιτρέπει την ευρύτερη και εις βάθος προσέγγιση των προβλημάτων του πελάτη.

Γλώσσες

Δουλεύω θεραπευτικά στα Ισπανικά (μητρική γλώσσα), Ελληνικά (έχοντας ζήσει στην Ελλάδα επί 11 χρόνια), καθώς στα Αγγλικά και Γερμανικά.

Πώς λειτουργεί;

Προσφέρω θεραπεία προσωπικώς στη διεύθυνση Forststr. 144, Στουτγάρδη, Γερμανία (βλ. Επικοινωνία) Contact), καθώς και on-line μέσω Skype ή τηλεφωνικώς.

Η διάρκεια μιας συμβουλευτικής/ ψυχοθεραπευτικής συνεδρίας είναι 60 λεπτά. Η διάρκεια της θεραπείας κυμαίνεται από λίγες συνεδρίες έως χρονικό διάσημα πέραν του έτους. Κάθε συνεδρία κοστίζει 40 €. Οι συνεδρίες είναι συνήθως εβδομαδιαίες, εκτός όταν σε οξείες καταστάσεις ή επιδείνωση των προβλημάτων χρειάζεται μεγαλύτερη συχνότητα συνεδριών. Η εμπιστευτικότητα είναι εξασφαλισμένη.

Μπορείτε να επικοινωνήσετε μαζί μου μέσω e-mail, Skype ή τηλεφωνικώς για την πρώτη συνάντηση (διαπροσωπική ή on-line). Η πρώτη συνεδρία είναι δωρεάν. Θα συζητήσουμε κυρίως το τι σας έχει φέρει στη θεραπεία, και εάν μπορώ να σας βοηθήσω, ή τα προβλήματά σας είναι πέραν της αρμοδιότητάς μου (π.χ. δεν δουλεύω με ψυχιατρικούς ασθενείς ή με παιδιά)

Εφόσον αποφασίσετε ότι θέλετε να δουλέψουμε μαζί, θα καθορίσουμε το πλαίσιο, δηλαδή την ώρα, τη συχνότητα και τον τρόπο, … και θα ξεκινήσουμε!
